About the WeatherApp

This Weather app is developed with OpenWeather API. The backend of the application is developed in SpringBoot and the frontend is developed in HTML, CSS and Javascript.


  • The Application is fetching the weather data based upon the location either provided by user or it autodetect the location with the location permission of the browser.

  • With location being entered the application is giving the current weather data and 5 days/3 hours forecast data from openweather API.

  • The current location is being fetched by the browser as Latitude and Longitude. The Latitude and Longitude is given to https://nominatim.openstreetmap.org/reverse?lat='+lat+'&lon='+lng+'&format=json API to convert it into known city/location.
